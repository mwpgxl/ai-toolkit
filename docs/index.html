AI Toolkit

Claude Code Skills / MCP Servers / AI Agent 工具目录

API Endpoints

GET /api/tools.json
完整工具目录（含 meta 和所有工具数据）
GET /api/categories.json
按分类聚合的索引（分类名、数量、工具列表）

Quick Start

# 获取完整工具目录
curl https://mwpgxl.github.io/ai-toolkit/api/tools.json

# 用 jq 筛选 MCP 服务器
curl -s https://mwpgxl.github.io/ai-toolkit/api/tools.json \
  | jq '.tools[] | select(.category=="mcp-server") | {name, stars, description}'

# 用 jq 按 Star 数排序前 10
curl -s https://mwpgxl.github.io/ai-toolkit/api/tools.json \
  | jq '[.tools[] | {name, stars, category}] | sort_by(-.stars)[:10]'
